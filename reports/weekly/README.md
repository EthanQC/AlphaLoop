# OpenClaw 周报

本目录只保留中文周报。历史英文正文已清理，后续报告由 `apps/openclaw-config/scripts/scheduled-report.mjs weekly prepare` 生成。

## 交付规则

- 生成时间：每周一 19:00（Asia/Shanghai）。
- 交付时间：每周一 20:00（Asia/Shanghai）。
- 窗口：上周四 20:00 到本周一 20:00，用来衔接周二到周四日报窗口；20:00 只交付已生成文件，不在交付时重新计算。
- 数据来源：SQLite 中的交易事实、审批编辑、规则提案、模拟盘持仓、偏好快照，以及 Longbridge 的 QQQ 行情快照。

## 必含章节

- 本周结论
- 市场主线回顾
- QQQ 与美股风险温度
- 模拟盘与执行复盘
- 规则提案与策略学习
- 偏好/策略变化
- 下周跟踪
- 需要人工确认事项
