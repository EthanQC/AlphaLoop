# OpenClaw 日报

本目录只保留中文日报。历史英文正文已清理，后续报告由 `apps/openclaw-config/scripts/scheduled-report.mjs daily prepare` 生成。

## 交付规则

- 生成时间：每周二、三、四 19:00（Asia/Shanghai）。
- 交付时间：每周二、三、四 20:00（Asia/Shanghai）。
- 窗口：上一交付日 20:00 到本交付日 20:00，使用连续窗口；20:00 只交付已生成文件，不在交付时重新计算。
- 固定观察：日报只固定分析 QQQ，不做泛关注列表。
- 数据来源：SQLite 中的交易事实、审批编辑、规则提案、模拟盘持仓，以及 Longbridge 的 QQQ 行情快照。

## 必含章节

- 今日结论
- 信息检索与分类
- 执行与模拟盘
- QQQ 固定观察
- 当前持仓
- 风险与异常
- 规则提案摘要
- 需要人工确认事项
