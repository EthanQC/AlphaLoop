样张 A

盯盘室终端

投资晨报

暖白纸面、宋体刊头带期号、日报即头版头条、提问是「向编辑部去信」。气质：私人财经报馆，重阅读体验。适合把它当每日读物的心态。
样张 C

作战室

浅色高密度 bento 网格、每张卡右上角自带数据新鲜度标签（把「诚实标注」变成视觉语言）、系统状态小卡常驻。气质：指挥中心，一屏尽收。
